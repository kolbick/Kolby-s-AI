Why Kolby built this cockpit

A deliberate rebuild inspired by OpenWebUI, wired to the same backend you chat with on the main page.

Back to chat

Built with intent

Kolby spent months experimenting with flows that therapists and founders actually use. The new UI trims the clutter and keeps focus on chat, models, and the Changing Tides proposal generator.

Backend stays the same

Every request still hits the configured OpenWebUI URL with the JWT stored in this repo. No new secrets, no dummy endpoints—just a cleaner coat of paint on the same live brain.

Mobile friendly

The slide-in drawer, sticky composer, and larger hit targets fix the tap blockers that haunted the old layout. It’s ready for iPhone thumbs.
